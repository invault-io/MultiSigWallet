# 以太坊多签智能合约

## 关于

适用于冷热钱包的多签智能合约. 

## 本合约的一些特性

1. 可用作任意n-of-m多签模式.
2. 支持ETH及其ERC20代币.
3. 防重放攻击. 
4. 主流硬件钱包支持(Trezor,Ledger).
5. 交易日志.
